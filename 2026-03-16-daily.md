## 📄 Top Hugging Face Papers (Trending AI/Agent Research)

### 1. Visual-ERM: Reward Modeling for Visual Equivalence
- **Authors:** Ziyu Liu, Shengyuan Ding, Xinyu Fang, Xuanlang Dai, Penghui Yang, Jianze Liang, Jiaqi Wang, Kai Chen, Dahua Lin, Yuhang Zang
- **URL:** https://huggingface.co/papers/2603.13224
- **Abstract:** Vision-to-code tasks require models to reconstruct structured visual inputs (charts, tables, SVGs) into executable representations with high visual fidelity. The authors propose Visual-ERM, a multimodal generative reward model that provides fine-grained, interpretable feedback for vision-to-code quality evaluation in the rendered visual space. Integrated into RL, Visual-ERM improves Qwen3-VL-8B-Instruct by +8.4 on chart-to-code. The work introduces VisualCritic-RewardBench for evaluating fine-grained image-to-image discrepancies on structured visual data.
- **Relevance:** Critical for agents that need to interpret and generate visual content, UI code, or structured documents.

### 2. LMEB: Long-horizon Memory Embedding Benchmark
- **Authors:** Xinping Zhao, Xinshuo Hu, Jiaxin Xu, Danyu Tang, Xin Zhang, Mengjia Zhou, Yan Zhong, Yao Zhou, Zifei Shan, Meishan Zhang, Baotian Hu, Min Zhang
- **URL:** https://huggingface.co/papers/2603.12572
- **Abstract:** Memory embeddings are crucial for memory-augmented systems like OpenClaw, but their evaluation is underexplored. LMEB introduces a comprehensive framework evaluating embedding models on long-horizon memory retrieval across 22 datasets and 193 zero-shot tasks spanning 4 memory types: episodic, dialogue, semantic, and procedural. Results show that traditional passage retrieval benchmarks (MTEB) exhibit orthogonality with long-horizon memory tasks—performance doesn't generalize.
- **Relevance:** Directly applicable to agentic systems requiring long-term memory, context retention, and retrieval-augmented generation.

### 3. Cheers: Decoupling Patch Details from Semantic Representations
- **Authors:** Yichen Zhang, Da Peng, Zonghao Guo, Zijian Zhang, Xuesong Yang, Tong Sun, Shichu Sun, Yidan Zhang, Yanghao Li, Haiyan Zhao, Wang Xu, Qi Shi, Yangang Sun, Chi Chen, Shuo Wang, Yukun Yan, Xu Han, Qiang Ma, Wei Ke, Liang Wang, Zhiyuan Liu, Maosong Sun
- **URL:** https://huggingface.co/papers/2603.12793
- **Abstract:** Cheers presents a unified multimodal model that decouples patch-level details from semantic representations, enabling both visual comprehension and generation in a single architecture. Key innovations include: (i) unified vision tokenizer with 4× token compression, (ii) LLM-based Transformer unifying autoregressive and diffusion decoding, (iii) cascaded flow matching head for detail refinement. Outperforms Tar-1.5B on GenEval and MMBench with only 20% training cost.
- **Relevance:** Enables more efficient multimodal agents capable of both understanding and generating visual content.
