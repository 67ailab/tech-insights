## 📄 Key Research Papers

### AIDev: Studying AI Coding Agents on GitHub
**arXiv: 2602.09185** | Feb 9, 2026  
A landmark dataset of **932,791 agent-authored pull requests** from five coding agents (OpenAI Codex, Devin, GitHub Copilot, Cursor, Claude Code) across 116,211 repositories and 72,189 developers. This is the first large-scale empirical study of how AI coding agents are actually being used in production. Key finding: agent PRs are now a measurable fraction of open-source development.  
🔗 https://arxiv.org/abs/2602.09185

### Security Threat Modeling for AI-Agent Protocols (MCP, A2A, Agora, ANP)
**arXiv: 2602.11327** | Feb 11, 2026  
The first systematic security analysis of four emerging agent communication protocols. Identifies **12 protocol-level risks** across creation, operation, and update phases. Includes a measurement-driven case study on MCP showing risks of wrong-provider tool execution. Critical reading for anyone building multi-agent systems.  
🔗 https://arxiv.org/abs/2602.11327

### Agentifying Agentic AI (AAAI 2026 Bridge Program)
**arXiv: 2511.17332** | Updated Feb 10, 2026  
Argues that true "agentic AI" requires more than autonomy — it needs explicit models of cognition, cooperation, and governance drawn from the multi-agent systems (AAMAS) community. Proposes bridging BDI architectures, mechanism design, and institutional modeling with modern LLM-based agents for transparent, accountable systems.  
🔗 https://arxiv.org/abs/2511.17332

## 💡 Key Takeaways

1. **Agent coding is now measurable at scale.** The AIDev dataset (932K agent PRs) proves AI coding agents are no longer experimental — they're a production reality across 100K+ repos.

2. **Agent security is lagging behind adoption.** Two separate papers this week highlight that agent communication protocols (MCP, A2A) and agent deployments lack mature security frameworks. The threat surface is expanding faster than defenses.
