### 2. S2D2: Fast Decoding for Diffusion LLMs via Training-Free Self-Speculation
- **Authors:** Ligong Han, Hao Wang, Han Gao, et al. (Red Hat AI)
- **URL:** https://huggingface.co/papers/2603.25702
- **Upvotes:** 2
- **Abstract:** Presents S2D2, a training-free self-speculative decoding framework for block-diffusion language models. Uses the same pretrained model as both drafter and verifier by reducing block size to one. Achieves up to 4.7x speedup over autoregressive decoding on SDAR, and 1.57x over tuned dynamic decoding baseline while improving accuracy by up to 4.5 points.

### 3. MemMA: Coordinating the Memory Cycle through Multi-Agent Reasoning and In-Situ Self-Evolution
- **Authors:** Minhua Lin, Zhiwei Zhang, Hanqing Lu, et al.
- **URL:** https://huggingface.co/papers/2603.18718
- **Upvotes:** 0
- **Abstract:** Proposes MemMA, a plug-and-play multi-agent framework that coordinates the memory cycle in memory-augmented LLM agents. Features a Meta-Thinker for structured guidance, Memory Manager for construction, and Query Reasoner for iterative retrieval. Introduces in-situ self-evolving memory construction that synthesizes probe QA pairs and converts failures into repair actions. Outperforms existing baselines on LoCoMo across multiple LLM backbones.

### 4. Revisiting On-Policy Distillation: Empirical Failure Modes and Simple Fixes
- **Authors:** Yuqian Fu, Haohuan Huang, Kaiwen Jiang, et al. (Institute of Automation, CAS)
- **URL:** https://huggingface.co/papers/2603.25562
- **Upvotes:** 0
- **Abstract:** Revisits on-policy distillation (OPD) for LLM post-training, identifying three failure modes: imbalanced one-token signal, unreliable teacher guidance on student-generated prefixes, and tokenizer/special-token mismatch distortions.
