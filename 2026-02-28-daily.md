## 🔥 Hacker News Highlights

- **"Agentic AI Is Neither Intelligent nor an Agent"** — Provocative essay sparking debate on whether current "agents" deserve the label. Top comment: "When you lose the ability to write, you lose the ability to think." ([HN](https://news.ycombinator.com/item?id=47136951))

- **"A Development Methodology for the Agentic AI Era"** — Argues that pre-AI methodologies like TDD/BDD need rethinking now that agents write the code in between input→output. Where does human expertise actually belong? ([HN](https://news.ycombinator.com/item?id=47115067))

- **Leviathan: AI Agents Write Laws and Govern Themselves** — A persistent HTTP world where LLM agents register, propose laws, and vote. Three agents running continuously — one already formed a coalition. Wild experiment in agent governance. ([HN](https://news.ycombinator.com/item?id=47177195))

- **EloPhanto: Self-Building Local AI Agent** — Show HN project where the agent builds its own tools when it lacks them. Claims 107+ self-built tools via a research→design→implement→test→deploy loop. ([HN](https://news.ycombinator.com/item?id=47157981))

- **"Sandboxes Won't Save You"** — Security discussion on agent containment. Key takeaway: sandboxes help but you must prevent write access to files executed outside the sandbox. ([HN](https://news.ycombinator.com/item?id=47154803))

- **AI Agents Making Data Portability the Default** — Thesis that agents demanding interoperability are finally breaking walled gardens, forcing platforms to let users govern their own data flow. ([HN](https://news.ycombinator.com/item?id=47157861))

## 📄 Notable Papers (arXiv, Feb 2026)

- **"Agents of Chaos"** (arXiv:2602.20021, Feb 23) — Landmark paper with 11 case studies of chaotic agent behavior: sensitive data exposure, server wipes, infinite loops, spoofing in realistic multi-tool environments. Essential reading for anyone deploying agents.

- **"AI Agents vs. Agentic AI: A Conceptual Taxonomy"** (arXiv:2505.10468) — Structured taxonomy distinguishing single AI agents from agentic AI systems, with actionable insights for designing next-gen systems.

- **ArchAgent** (arXiv:2602.22425, Feb 25) — Agentic AI applied to computer architecture discovery. Agents autonomously exploring hardware design spaces.

- **DualPath** (arXiv:2602.21548, Feb 25) — Tackles the storage bandwidth bottleneck in agentic LLM inference. Critical infrastructure work as agents scale up.

- **"Toward an Agentic Infused Software Ecosystem"** (Feb 24) — Envisions the "Dark Factory" where agents independently create entire applications from high-level specs. References the "five levels from spicy autocomplete to software factory."

- **"Codified Context: Infrastructure for AI Agents"** (Feb 24) — Framework for giving agents structured context in complex codebases, with MCP retrieval servers and constitution documents.
