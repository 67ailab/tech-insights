[Paper](https://huggingface.co/papers/2505.10468) — Trending on HF Daily Papers

A comprehensive taxonomy distinguishing **AI Agents** (modular, task-specific LLM-driven systems) from **Agentic AI** (multi-agent collaboration with dynamic task decomposition, persistent memory, and orchestrated autonomy). Key contributions:
- **AI Agents**: Tool integration + prompt engineering + reasoning for narrow automation
- **Agentic AI**: Multi-agent collaboration, persistent memory, orchestrated autonomy
- Challenges mapped: hallucination, brittleness, emergent behavior, coordination failure
