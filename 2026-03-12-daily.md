## 📚 Top Hugging Face Papers (AI/Agents)

### 1. OpenClaw-RL: Train Any Agent Simply by Talking
- **Authors:** Yinjie Wang, Xuyang Chen, Xiaolong Jin, Mengdi Wang, Ling Yang
- **Upvotes:** 48
- **URL:** https://huggingface.co/papers/2603.10165
- **Abstract:** Every agent interaction generates a next-state signal (user reply, tool output, terminal/GUI state change), yet no existing agentic RL system recovers it as a live, online learning source. OpenClaw-RL enables agents to improve simply by being used, recovering conversational signals from user re-queries, corrections, and explicit feedback. The framework extracts evaluative signals (scalar rewards via PRM judge) and directive signals (via Hindsight-Guided On-Policy Distillation) from next-state signals across personal conversations, terminal executions, GUI interactions, SWE tasks, and tool-call traces.
- **Code:** https://github.com/Gen-Verse/OpenClaw-RL

### 2. MA-EgoQA: Question Answering over Egocentric Videos from Multiple Embodied Agents
- **Authors:** Kangsan Kim, Yanlai Yang, Suji Kim, Woongyeong Yeo, Youngwan Lee, Mengye Ren, Sung Ju Hwang
- **URL:** https://huggingface.co/papers/2603.09827
- **Abstract:** First benchmark for question answering over multiple long-horizon egocentric videos from embodied agents (1,741 questions, 5 categories, 6 agents, 7 days). Proposes EgoMAS, a training-free baseline using shared memory and dynamic retrieval that outperforms frontier models like Gemini-2.5-Flash and GPT-5. Addresses the challenge of compressing and communicating high volumes of individual sensory inputs from multiple embodied agents collaborating in shared environments.

### 3. In-Context Reinforcement Learning for Tool Use in Large Language Models
- **Authors:** Yaoqi Ye, Yiran Zhao, Keyu Duan, Zeyu Zheng, Kenji Kawaguchi, Cihang Xie, Michael Qizhe Shieh
- **URL:** https://huggingface.co/papers/2603.08068
- **Abstract:** Proposes In-Context Reinforcement Learning (ICRL), an RL-only framework that eliminates the need for supervised fine-tuning (SFT) by leveraging few-shot prompting during rollout. ICRL introduces in-context examples within rollout prompts to teach models how to invoke external tools, gradually reducing examples until reaching zero-shot tool use. Achieves state-of-the-art performance across reasoning and tool-use benchmarks, demonstrating a scalable, data-efficient alternative to traditional SFT-based pipelines.

### 4. Can Large Language Models Keep Up? Benchmarking Online Adaptation to Continual Knowledge Streams (OAKS)
- **Authors:** Jiyeon Kim, Hyunji Lee, Dylan Zhou, Sue Hyun Park, Seunghyun Yoon, Trung Bui, Franck Dernoncourt, Sungmin Cha, Minjoon Seo
- **URL:** https://huggingface.co/papers/2603.07392
- **Abstract:** Introduces OAKS benchmark for evaluating models' online adaptation to streaming, continually updating knowledge. Real-world knowledge evolves constantly—can LLMs adapt on the fly? Evaluating 14 models, both state-of-the-art models and agentic memory systems fail to adapt robustly, demonstrating delays in state-tracking and susceptibility to distraction within streaming environments.

### 5. RetroAgent: From Solving to Evolving via Retrospective Dual Intrinsic Feedback
- **Authors:** Xiaoying Zhang, Zichen Liu, Yipeng Zhang, Xia Hu, Wenqi Shao
- **URL:** https://huggingface.co/papers/2603.08561
- **Abstract:** Introduces RetroAgent, an online RL framework empowering agents to master complex interactive environments by evolving, not just solving. Features hindsight self-reflection producing dual intrinsic feedback: (1) numerical feedback tracking incremental subtask completion, and (2) language feedback distilling reusable lessons into memory. Outperforms GRPO-trained agents by +18.3% on ALFWorld, +15.4% on WebShop, +27.1% on Sokoban, and +8.9% on MineSweeper.
- **Code:** https://github.com/zhangxy-2019/RetroAgent

## 🔑 Key Trends & Insights

### 1. **Agentic Security is the New Frontier**
The McKinsey AI platform hack (439 points on HN) underscores a critical reality: AI agents introduce novel attack surfaces while inheriting classic vulnerabilities. As agentic AI deployments accelerate, security teams must treat agents as both potential victims and attack vectors. The convergence of AI autonomy with traditional injection vulnerabilities creates compound risks.

### 2. **Online Learning for Agents is Maturing**
OpenClaw-RL represents a paradigm shift—agents that learn continuously from interaction rather than batch training. The insight that "next-state signals are universal" across conversations, terminals, GUIs, and tool calls suggests a unified theory of agentic learning. This could dramatically reduce the cost of agent customization and improvement.

### 3. **Embodied Multi-Agent Systems Need Shared Memory**
MA-EgoQA highlights the emerging challenge of coordinating multiple embodied agents in shared spaces. The fact that a training-free method (EgoMAS) outperforms frontier models suggests that architecture and memory design matter more than raw model scale for multi-agent coordination.

### 4. **Tool Use Without SFT is Now Viable**
ICRL demonstrates that reinforcement learning alone, with clever in-context prompting, can teach tool use without expensive supervised fine-tuning. This lowers the barrier for organizations to deploy specialized agents without massive labeled datasets.

### 5. **Current Agents Struggle with Continual Knowledge**
The OAKS benchmark reveals a sobering gap: neither frontier models nor agentic memory systems handle streaming knowledge well. As real-world information evolves continuously, this limitation will increasingly constrain agent reliability in dynamic domains.

### 6. **Experiential Learning via Self-Reflection**
RetroAgent's dual feedback mechanism (numerical + language) points toward agents that explicitly learn from failures, not just optimize for success. The memory buffer with SimUtil-UCB retrieval suggests agents are developing something akin to episodic memory.
