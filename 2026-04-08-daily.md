## 📄 Top Hugging Face Papers (AI/Agents)

### 1. Claw-Eval: Toward Trustworthy Evaluation of Autonomous Agents
- **Authors:** Bowen Ye, Rang Li, Qibin Yang, et al. (Peking University, HKU)
- **URL:** https://arxiv.org/abs/2604.06132
- **Abstract:** Introduces Claw-Eval, an end-to-end evaluation suite for autonomous agents addressing three critical gaps: (1) trajectory-opaque grading, (2) underspecified safety/robustness evaluation, and (3) narrow modality coverage. Features 300 human-verified tasks across 9 categories with full-trajectory auditing through execution traces, audit logs, and environment snapshots. Experiments on 14 frontier models reveal trajectory-opaque evaluation misses 44% of safety violations and 13% of robustness failures.

### 2. Learning to Retrieve from Agent Trajectories (LRAT)
- **Authors:** Sunhao Dai, Changle Qu, Liang Pang, et al. (Renmin University, CAS)
- **URL:** https://arxiv.org/abs/2604.04949
- **Abstract:** Proposes learning to retrieve from agent trajectories as a new training paradigm for agentic search. Identifies key behavioral signals from agent trajectories: browsing actions (necessary for success), unbrowsed rejections (reliable negatives), and post-browse reasoning traces (relevance intensity indicator). LRAT framework mines supervision from agent trajectories without human annotation, consistently improving evidence recall and task success across diverse agent architectures.

### 3. Beyond Accuracy: Unveiling Inefficiency Patterns in Tool-Integrated Reasoning
- **Authors:** Qisheng Su, Shiting Huang, Zhen Fang, et al. (USTC, Shanghai Innovation Institute)
- **URL:** https://arxiv.org/abs/2604.05404
- **Abstract:** Introduces PTE (Prefill Token Equivalents), a hardware-aware efficiency metric for Tool-Integrated Reasoning (TIR) that accounts for KV-Cache eviction and long tool responses. Validates PTE against wall-clock latency (r=0.9253) in high-concurrency settings. Identifies four inefficiency patterns: Confirmatory Tool Usage, Tool-Mixing, Lack of Tool Priors, and Tool Format Collapse. Finds negative correlation between PTE costs and reasoning correctness—more tools doesn't mean better answers.

### 4. Video-MME-v2: Towards the Next Stage in Benchmarks for Comprehensive Video Understanding
- **Authors:** Multiple contributors
- **URL:** https://arxiv.org/abs/2604.05015
- **Abstract:** Introduces Video-MME-v2, a comprehensive benchmark for video MLLMs with progressive tri-level hierarchy (visual information aggregation → temporal dynamics → complex reasoning) and group-based non-linear evaluation. 800 videos, 3,200 questions, 3,300 human-hours. Best model (Gemini-3-Pro) scores 49.4 vs human experts at 90.7, revealing substantial capability gaps.

### 5. ACES: Who Tests the Tests? Leave-One-Out AUC Consistency for Code Generation
- **Authors:** Hui Sun, Yun-Ji Zhang, Zheng Xie, et al. (Nanjing University)
- **URL:** https://arxiv.org/abs/2604.03922
- **Abstract:** Addresses circular dependency in LLM-generated code/test evaluation. Proposes LOO-AUC (Leave-One-Out AUC) to measure test discriminative power without knowing code correctness. ACES framework provides closed-form weights (ACES-C) and optimized weighting (ACES-O), achieving SOTA Pass@k on HumanEval, HumanEval+, and MBPP using only binary pass matrix.

## 🔑 Key Trends & Insights

### 1. Agent Evaluation Maturation
Multiple frameworks (Claw-Eval, Video-MME-v2) emphasize **trajectory-level auditing** over output-only grading. The field is moving toward evaluating *how* agents accomplish tasks, not just *what* they produce. Safety and robustness are now embedded in normal workflow evaluation rather than isolated red-teaming.

### 2. Efficiency Awareness in Agentic Systems
PTE metric reveals that **more tool use ≠ better performance**. Trajectories with higher PTE costs correlate with lower correctness. This challenges the "scale tool use" approach and suggests selective, high-quality tool invocation is preferable to exhaustive tool exploration.

### 3. Agent-Centric Retrieval Training
LRAT demonstrates that retrieval models trained on **agent interaction data** outperform human-centric retrievers for agentic search. Agent trajectories provide naturally abundant supervision signals (browsing actions, reasoning traces) that better align with how agents consume search results.

### 4. Security Focus for AI Systems
Anthropic's Project Glasswing and Claude Mythos cybersecurity assessment signal growing industry attention to **AI supply chain security**. As agents gain autonomy, securing the software they interact with becomes critical infrastructure.

### 5. Open-Source Agent Tooling
Google's Scion testbed and Apple Silicon fine-tuning frameworks lower barriers for **agent experimentation**. The ecosystem is maturing with standardized evaluation environments and accessible training infrastructure.
