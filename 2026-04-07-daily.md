## 📚 Top Hugging Face Papers (AI/Agents)

### 1. OpenWorldLib: A Unified Codebase and Definition of Advanced World Models
- **Authors:** Bohan Zeng et al. (Peking University, Kling Team, Tsinghua, HKUST, NUS)
- **URL:** https://huggingface.co/papers/2604.04707
- **Abstract:** Introduces OpenWorldLib, a standardized framework for Advanced World Models. Proposes unified definition: world models are perception-centered frameworks with interaction and long-term memory for understanding/predicting the complex world. Integrates models across tasks enabling efficient reuse and collaborative inference. Key modules: Operator, Synthesis (visual/audio), Reasoning, Representation, Memory, and Pipeline.

### 2. MinerU2.5-Pro: Pushing the Limits of Data-Centric Document Parsing at Scale
- **Authors:** Yuan Qu, Mengzhang Cai et al. (Shanghai AI Lab, Peking University)
- **URL:** https://huggingface.co/papers/2604.04771
- **Abstract:** Advances document parsing through data engineering alone, keeping 1.2B architecture fixed. Data Engine features: Diversity-and-Difficulty-Aware Sampling (10M→65.5M samples), Cross-Model Consistency Verification, Judge-and-Refine pipeline. Three-stage training: pre-training, hard sample fine-tuning, GRPO alignment. Achieves 95.69 on OmniDocBench v1.6, +2.71 over baseline, surpassing models with 200× more parameters.

### 3. TriAttention: Efficient Long Reasoning with Trigonometric KV Compression
- **Authors:** Xi Lin, Wei Huang et al.
- **URL:** https://huggingface.co/papers/2604.04921
- **Abstract:** Addresses KV cache memory bottlenecks in extended LLM reasoning. Discovers Q/K concentration in pre-RoPE space—vectors concentrate around fixed non-zero centers stable across positions. TriAttention leverages trigonometric series from these centers to score key importance. On AIME25 with 32K tokens: matches Full Attention accuracy with 2.5× higher throughput or 10.7× KV memory reduction. Enables long-context deployment on consumer GPUs.

### 4. AURA: Always-On Understanding and Real-Time Assistance via Video Streams
- **Authors:** Xudong Lu, Yang Bo et al. (Huawei Research, CUHK MMLab)
- **URL:** https://huggingface.co/papers/2604.04184
- **Abstract:** End-to-end streaming VideoLLM framework for continuous video processing with real-time QA and proactive responses. Addresses limitations of decoupled trigger-response pipelines. Features: Interactive Video Stream Context Management, Coarse-to-Fine Data Engine (5-stage pipeline for Real-Time/Proactive/Multi-Response QA), Silent-Speech Balanced Loss, Real-time Streaming Inference with KV-cache reuse. Runs at 2 FPS on two 80G accelerators with ASR/TTS.

### 5. LIBERO-Para: A Diagnostic Benchmark for Paraphrase Robustness in VLA Models
- **Authors:** Research team (arXiv:2603.28301)
- **URL:** https://huggingface.co/papers/2603.28301
- **Abstract:** Benchmark and metrics for evaluating paraphrase robustness in Vision-Language-Action (VLA) models. Introduces PRIDE (Paraphrase Robustness Index in Robotic Instructional DEviation) metric. Key findings: paraphrase fragility persists across architectures/data scales; object grounding is primary bottleneck; failures are predominantly planning-level not execution-level. Critical for improving robot instruction following.

## 🔍 Key Trends & Insights

### 1. **Agentic Infrastructure Maturing Rapidly**
Freestyle's sandbox forking (400ms, memory-preserving) and Ghost Pepper's local voice agents show the infrastructure layer for agentic systems is becoming production-ready. Sub-second state preservation enables truly interruptible/resumable agent workflows.

### 2. **Long-Context Reasoning Breakthrough**
TriAttention's 10.7× KV compression while maintaining accuracy is significant for agentic systems that require long-horizon planning. This directly addresses the "context window tax" that has limited agent autonomy.

### 3. **World Models as Agent Foundation**
OpenWorldLib's unified framework suggests the field is converging on world models as essential for embodied agents. Perception + interaction + memory = foundation for agents that understand environments, not just react.

### 4. **Real-Time Streaming Agents**
AURA's always-on video understanding with proactive responses represents the next evolution beyond query-response agents. Continuous observation + autonomous decision-making = more natural human-agent collaboration.

### 5. **Claude Code Pain Points Reflect Growing Pains**
The massive engagement on Claude Code usability issues (1,154 points, 632 comments) reveals the gap between agent hype and real engineering workflows. Complex, multi-file tasks remain challenging—agents excel at scaffolding but struggle with sustained complex work.

### 6. **Data-Centric > Architecture-Centric**
MinerU2.5-Pro's results (beating 200× larger models via data engineering alone) reinforce that for domain-specific agent tasks, curated training data may matter more than model scale.

### 7. **Governance Debate Intensifying**
The New Yorker piece on Altman (1,524 points) shows public discourse is shifting from "can we build AGI?" to "who should control it?" This has direct implications for open-source vs. closed agent development.
