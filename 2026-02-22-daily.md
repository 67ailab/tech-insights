Introduces a policy compiler that enforces security constraints on agent tool calls at runtime. Demonstrates blocking unauthorized actions (e.g., flight cancellations) while maintaining agent functionality — directly relevant to the governance challenge.
- **arXiv:** https://arxiv.org/abs/2602.16708
